GEMS.Population — Type
Population

A Type for a simple population. Acts as a container for a collection of individuals.

Fields

  • individuals::Vector{Individual}: List of associated individuals
  • maxage: Age of the oldest individual

each!(f, population::Population)

Applies function f to all individuals in the population.

Example

each!(i -> i.age = i.age + 1, pop) lets all individuals in the populaiton pop age by one year.

GEMS.group_by_age — Method
group_by_age(df::DataFrame)

Groups a dataframe by its "age" column. The column "sum" contains the sum of each individual in this age. This also adds rows for each age from 0 till the maximum age in the "age" column.

Returns

  • returns a DataFrame with an additional column "sum". This column contains the count of the age in the DataFrame.
